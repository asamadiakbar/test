--------- Git Hub ---------

1. Create Username and Email : git config --globel user.name yourname/email
2. Check User and email: git config --globel user.yourname/email
3. git initialized(current directory) : git init
4. Create a new file : touch filename.xxx (like: touch contact.php)
5. Check file status : git status
6. push fil in commit area : git add -A
7. pass commit message : git commit -m "your massage"
8. To get your file back if someone edits or deletes your file without your permission : git checkout -f
9. Check commit log : git log
10. Filter commit log : git log -p -1
11. Check your changes(working file to staging area) : git diff
12. Check your changes(working area to last commit) : dit diff --staged
13. Push file without commit : git commit -a -m "Skipped Staging area."
14. delete your file in staging area : git rm --cached filename.txt
15. Delete you file staging and working direcotory area : git rm --cached filename.php -f
16. Create new Branch : git branch branch name
17. Change Branch : git checkout branch name 
18. Merge Branch : git merge branchname
19. Add Remote : git remote add origin https://github.com/akbarwebzlabz/Test.git
20. push data : git push -u origin branchname. 
21. multiple account generate ssh key : ssh-keygen -t ed25519 -C "akbarhusainasamadi@gmail.com" 
22. single account generate ssh key : ssh-keygen -t rsa -b 4096 -C "akbarhusainasamadi@gmail.com"
